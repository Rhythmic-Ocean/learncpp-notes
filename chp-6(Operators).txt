6.7
NOTE: For floating point comparision, since floating point storage can be imprecise, doing:
    float a = 1.4f;
    float b = 0.3f + 1.1f;
    bool val = a == b;
    the bool val will prob not be true. So better is to create a custom function that checks for equal stuffs:

    Something like this:
    #include <algorithm> ->for std::max
    #invlude <cmath> -> for std::abs
    bool approximatelyEqualRel(double a, double b, double relEpsilon)
    {
      return (std::abs(a - b) <= (std::max(std::abs(a), std::abs(b)) * relEpsilon)); -> calculates the positive value for difference between a and b and as long as it's less than 
                          relEpsilon% or less than the max of a or b, it's equal. So if a and b are 1 and 1.000001 and relEpsilon is 0.1% i.e 0.001, then 1-1.000001 = 0.000001 which
                          is less than 1.000001*0.001, therefore they are considered equal
    }


    just negate this to get != equivalent

    == is only acceptable for comparision between floating point literals like 1.1 == 1.1, which seems quite useless
    or
    float a = 1.1f;
    float b = 1.1f;
    a == b; might work
    however even the above function breaks at numbers close to zero cuz:
      for comparision between a =  0 and b = 0.0000000001, we might demand rel difference required to be only 1%, but 1% of 0.000000001 will be < b-0, however, the answer would have
      been different if it was between a = 1 and b = 1.00000001 with rel episelon of 1%. So better if we use mix of absolute and relative episolen with a function like

    bool approxAbsRel(double a, double b, double relEpsilon, double absEpsilon){
      bool truth = std::abs(a - b) < absEpsilon;
      if (truth) return True;
      return approximatEqualRel(a, b, relEpsilon);
    }

    NOTE: the above 2 functions can be made constexpr in c++23 but before this std::abs() was not made a constexpr function so will show error

    A workaround is a custom constexpr function that does abs stuff.
    templete <typename T>
    constexpr constAbs T (T x){
        return x < 0? -x : x;
      }
